## Common Workflows

### Adding a New Lead

1. Navigate to **Customer**
2. Click **"+ New Lead"**
3. Enter: Name, Contact, Source
4. Click **"Save Lead"**

**Keyboard Shortcut:** `Ctrl+K` → "Add new lead"

---

### Creating a Sales Proposal

1. Go to **New Business**
2. Click **"Start New Proposal"**
3. Select/Create customer
4. Complete **Fact Finding**
5. Review **FNA Results**
6. Select **Products** (or use AI recommendations)
7. Generate **Quotation**
8. Complete **Application**

**Keyboard Shortcut:** `Ctrl+K` → "Create new proposal"

---

### Generating a Quick Quote

1. Navigate to **Products**
2. Find desired product
3. Click **"Quick Quote"**
4. Enter: Age, Coverage, Payment frequency
5. View instant premium

**Alternative:** `Ctrl+K` → "Quick quote for [product]"

---

### Adding a Task

1. Go to **To Do**
2. Click **"+ New Task"**
3. Enter: Title, Due Date, Priority
4. Optionally link to customer
5. Click **"Save Task"**

**Keyboard Shortcut:** `Ctrl+K` → "Add task [description]"

---

### Using Mira AI

**To Open:**
- Click chat icon (bottom-right)
- Press `Ctrl+K` (Windows) or `Cmd+K` (Mac)

**Example Commands:**
- "Show me my customers"
- "What are my sales this month?"
- "Create a task to follow up with John"

## Status Indicators

### Lead Statuses

| Status | Meaning | Next Action |
|--------|---------|-------------|
| **Not Contacted** | New lead | Make first contact |
| **Contacted** | Initial contact made | Qualify interest |
| **Qualified** | Shows interest | Create proposal |
| **Proposal** | Proposal sent | Follow up |
| **Negotiation** | Discussing terms | Address objections |
| **Won** | Converted to customer | Submit application |
| **Lost** | Did not convert | Document reason |

### Task Priorities

| Priority | Color | When to Use |
|----------|-------|-------------|
| **High** | Red | Urgent, immediate action |
| **Medium** | Yellow | Important, scheduled |
| **Low** | Gray | Can wait, non-urgent |

### Application Statuses

| Status | Meaning | Action Required |
|--------|---------|-----------------|
| **Draft** | Not submitted | Complete application |
| **Pending Review** | Submitted | Wait for underwriting |
| **Underwriting** | Being evaluated | Provide additional info if requested |
| **Medical Required** | Needs medical exam | Schedule exam for customer |
| **Approved** | Accepted | Proceed to issuance |
| **Issued** | Policy active | Deliver policy documents |
| **Declined** | Not approved | Inform customer, document reason |

## Common Issues & Solutions

### Cannot Login
**Solution:**
1. Verify username/password
2. Check internet connection
3. Try "Forgot Password"
4. Contact support if 2FA issues

### Page Not Loading
**Solution:**
1. Refresh browser (F5)
2. Clear cache and cookies
3. Try different browser
4. Check system status page

### Data Not Saving
**Solution:**
1. Check required fields
2. Verify internet connection
3. Try again in a few minutes
4. Contact support if persistent

### Cannot Find Customer
**Solution:**
1. Check spelling
2. Try partial name
3. Search by phone/email
4. Use advanced filters
5. Check if deleted

## Emergency Procedures

### System Outage
1. Check system status page
2. Wait for automatic recovery
3. Use backup (if available)
4. Contact support if extended
5. Document affected work

### Data Error
1. Stop making changes
2. Document the error
3. Take screenshots
4. Contact support immediately
5. Don't try to fix manually

### Lost Access
1. Try password reset
2. Check 2FA device
3. Contact administrator
4. Verify account status
5. Request access restoration
