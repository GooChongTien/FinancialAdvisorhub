## Quick Decision Tree

```
┌─────────────────────────────────────────────────────────────┐
│ WHAT COLOR SHOULD I USE?                                   │
└─────────────────────────────────────────────────────────────┘

Is it a PRIMARY ACTION? (Save, Submit, Create)
  → Use primary-600 (Blue)

Is it a SUCCESS/POSITIVE indicator? (Revenue, Client, Approve)
  → Use green-600/700

Is it an ERROR or DESTRUCTIVE action? (Delete, Error, Expired)
  → Use red-600/700

Does it need ATTENTION? (Hot Lead, Warning, Pending)
  → Use orange-500/700

Is it INFORMATIONAL? (Tips, Proposal status)
  → Use yellow-500/700

Is it SECONDARY/NEUTRAL? (Cancel, Disabled, Structure)
  → Use slate-100/500/700

When in doubt? → Use primary blue for actions, gray for structure
```
